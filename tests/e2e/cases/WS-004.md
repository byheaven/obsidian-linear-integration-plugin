# WS-004 Reject Local Team Edits

## Purpose

Verify that `linear_team` and `linear_team_id` are display-only locally and cannot move an issue across teams.

## Setup

- WS-001 has already created one linked issue.

## Steps

1. Change `linear_team` locally to another workspace team.
2. Run sync.
3. Inspect logs and the remote issue.
4. Restore the original local team fields.

## Expected Results

- The plugin reports that team changes must be made in Linear.
- The remote issue team is unchanged.
- No cross-workspace reassignment occurs.

## Artifacts To Collect

- `dev:errors`
- console errors

## Cleanup

- Restore the original local team values before the suite continues.
