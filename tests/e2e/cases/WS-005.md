# WS-005 Sync Local Draft Comments To Linear

## Purpose

Verify that text added after the sync marker in `# New Comment` becomes exactly one Linear comment.

## Setup

- WS-001 has already created one linked issue per workspace.

## Steps

1. Insert a draft comment into the managed note body.
2. Run sync.
3. Fetch the remote issue comments.
4. Read the local note body again.

## Expected Results

- One new remote comment is created.
- The draft area is cleared locally.
- The sync marker timestamp advances.
- The mirrored comments section includes the new comment.

## Artifacts To Collect

- note snapshot before sync
- note snapshot after sync

## Cleanup

- Normal suite cleanup only.
