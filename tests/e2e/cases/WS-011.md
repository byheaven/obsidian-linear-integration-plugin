# WS-011 Show Sync Summary Notice

## Purpose

Verify that running the top-level Obsidian sync command shows a popup summarizing the sync result.

## Setup

- At least one linked issue note already exists for each enabled workspace.
- Obsidian desktop and CLI are connected to the target vault.

## Steps

1. Run the plugin command `Sync Linear issues` through Obsidian.
2. Capture the rendered notice text from the Obsidian UI.

## Expected Results

- The command emits a sync summary notice.
- The notice starts with `Linear Integration`.
- The notice includes `Created:`, `Updated:`, `Conflicts:`, and `Errors:` lines.
- The notice reflects the current sync result without requiring a direct `syncManager.syncAll()` call.

## Artifacts To Collect

- captured notice text
- `dev:errors`
