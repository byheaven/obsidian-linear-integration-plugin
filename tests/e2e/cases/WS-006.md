# WS-006 Mirror Remote Comments Back To Notes

## Purpose

Verify that remote Linear comments are rendered into `# Comments` and do not pollute the local draft area.

## Setup

- WS-001 has already created one linked issue per workspace.

## Steps

1. Add one remote comment through the Linear API helper.
2. Run sync.
3. Read the local note body.

## Expected Results

- The remote comment appears in `# Comments`.
- `# New Comment` stays reserved for local drafts.

## Artifacts To Collect

- post-sync note snapshot

## Cleanup

- Normal suite cleanup only.
